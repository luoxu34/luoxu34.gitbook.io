# Windows 10 Shortcut 

| 键1  | 键2         | 键3   | 作用                               |
| ---- | ----------- | ----- | ---------------------------------- |
| Win  | TAB         |       | 激活多任务视图                     |
| Win  | A           |       | 激活操作中心                       |
| Win  | C           |       | 通过语音激活Cortana                |
| Win  | D           |       | 显示桌面                           |
| Win  | E           |       | 打开文件管理器                     |
| Win  | I           |       | 打开Windows 10设置                 |
| Win  | K           |       | 激活无线显示器连接或音频设备连接   |
| Win  | L           |       | 锁定屏幕                           |
| Win  | P           |       | 投影屏幕                           |
| Win  | R           |       | 运行                               |
| Win  | S           |       | 激活Cortana                        |
| Win  | V           |       | 剪贴板历史记录                     |
| Win  | X           |       | 打开高级用户功能                   |
| Win  | .           |       | 打开Emoji面板                      |
| Win  | ;           |       | 打开Emoji面板                      |
| Win  | 上/下/左/右 |       | 移动应用窗口                       |
| Win  | 1/2/3       |       | 打开任务栏中固定的程序             |
| Win  | Ctrl        | D     | 创建一个新的虚拟桌面               |
| Win  | Ctrl        | F4    | 关闭最近使用的虚拟桌面             |
| Win  | Ctrl        | 左/右 | 切换虚拟桌面                       |
| Win  | Shift       | 左/右 | 将应用从一个显示屏移至另一个显示屏 |
| Win  | Shift       | S     | 自由截图模式                       |
